Gmail Keyword Deep Search – Installation & Usage Guide

Official Chrome Web Store page: Gmail Keyword Deep Search
We never store or share your Gmail data. Read our Privacy Policy.

Gmail Keyword Deep Search is a Chrome extension that helps you search deeply inside your Gmail inbox. It scans email metadata (subject, sender, recipients, labels, date) in real time, shows live updates in a full dashboard, and lets you filter, export, and open emails directly in Gmail.

1. Install in Chrome

  1. Open the extension page in the Chrome Web Store.
  2. Click Add to Chrome → Add Extension.
  3. Once installed, click the extension icon or open the new dashboard tab.

2. First-Time Setup

The first time you search, you’ll be asked to log in with Google:

  1. Click Search or use the Change Account button.
  2. Select your Gmail account and grant access.
  3. ⚠️ Google may show a warning that “This app isn’t verified.”
    Click Advanced → Go to Gmail Keyword Deep Search to continue. This appears only because the app is not yet verified by Google, but it is safe to use.

3. How to Use

  1. Enter a keyword (e.g. invoice) in the dashboard search bar.
  2. Click Search. The extension will start scanning your Gmail messages.
  3. Progress updates appear live in the Email Search Updates panel — showing the keyword and how many emails have been checked.
  4. You can filter results by labels, date ranges, or refine after search using the filter panel.
  5. Click an email row to preview metadata or redirect to Gmail to view the full email.
  6. Use the Export button to export filtered results as CSV.
  7. Use Stop to end a search midway, and Clear to reset results.

4. Settings

By default, the extension scans up to 10,000 emails per search. You can change this limit anytime from the Settings page.

5. Key Features

Full Dashboard UI
No more popup — a clean, full-page dashboard for searches, filters, and results.
Email Search Updates
Shows the keyword being searched and the live progress of emails being scanned.
Advanced Filters
Filter results by Gmail labels and date ranges, even after the search is complete.
Export Results
Export matched and filtered emails in CSV format for external use.
View in Gmail
Click any result to open the full email directly in Gmail.
Customizable Search Limit
Default 10,000 email scan limit, adjustable in settings.
Privacy
Only email metadata is stored locally; full email bodies are not stored at all.
