Gmail Keyword Deep Search – Installation & Usage Guide

We never store or share your Gmail data. Only lightweight metadata (IDs, headers, labels, dates) is saved locally. Privacy Policy.

Gmail Keyword Deep Search is a Chrome extension that lets you search your Gmail inbox with powerful filters and a modern dashboard UI.

1. Install in Chrome

You can install the extension directly from the Chrome Web Store.

  1. Open Chrome.
  2. Visit the extension’s page on the Chrome Web Store.
  3. Click Add to Chrome → Add Extension.
  4. The extension will appear in your toolbar once installed.

2. First-Time Setup

The extension uses Google OAuth via chrome.identity. The first time you run a search (or click Change Account):

  1. A Google sign-in window will appear asking you to select your Gmail account.
  2. Google will request permission to allow the extension to read your emails.
  3. Click Allow to grant access. This is required for scanning emails.
⚠️
Some users may see a warning: “This app hasn’t been verified.”
This happens because the extension is not yet verified by Google. It is safe to continue.

- Click Advanced
- Then click Go to Gmail Keyword Deep Search (unsafe)

3. How to Use

  1. Click the extension icon in your toolbar. This will open the full dashboard (not just a small popup).
  2. Enter a keyword (example: invoice) and click Search.
  3. Results will appear in real time as emails are scanned.
  4. Click any email to view its full content, or open it directly in Gmail.
  5. Use filters on the left panel to refine results by labels or date range.
  6. Use Stop to stop the crawl, Clear to reset, and Change Account to switch accounts.
  7. You can also change dashboard background themes from the settings menu.

4. Features

Modern Dashboard
Full-page dashboard instead of a small popup, with better navigation and filters.
Theme Customization
Change background themes for a personalized look.
Deep Scanning
Finds matches in subject, body text, and links.
Email Preview
View full email content directly in the dashboard or jump to Gmail.
Label & Date Filters
Narrow results by Gmail labels and custom date ranges.
No Body Storage
We do not store email bodies — only metadata is saved locally.
Live Progress
See the number of emails scanned in real time.
Deduplication
Unique results only — no duplicate sender entries.
